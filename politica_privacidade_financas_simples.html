Política de Privacidade – Finanças Simples

Desenvolvedor: ON Lab
Última atualização: 2025

O aplicativo Finanças Simples foi desenvolvido pela ON Lab com o objetivo de oferecer uma ferramenta simples para controle financeiro pessoal. Esta Política de Privacidade explica como tratamos seus dados.

1. Coleta de dados

O aplicativo não coleta informações pessoais sensíveis, como nome, e-mail, número de telefone ou identificadores diretos.

O app coleta apenas dados inseridos pelo usuário (ex.: despesas e receitas) e informações técnicas básicas para funcionamento.

2. Uso dos dados

Os dados são usados exclusivamente para funcionamento do app, exibição das informações financeiras e estabilidade do aplicativo.

3. Compartilhamento de dados

Não compartilhamos nenhum dado com terceiros.

4. Armazenamento

Todos os dados ficam armazenados localmente no dispositivo.

5. Segurança

Utilizamos práticas padrão de segurança local.
